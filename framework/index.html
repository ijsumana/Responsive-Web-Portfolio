Movie Trailer Display

A Python class has been created to instantiate my favorite movies with their title, story line, trailer youtube url and poster image. The list of these movies are then sent to a web application which displays all these movies in tile display. Clicking on the movie poster starts movie trailer display pop up from youtube. When a movie trailer is displayed the poster image background is colored yellow to keep track of the watched trailers. Mouse pointer over the movie poster displays the story line of the movie as tooltip.
